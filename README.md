# UnitTesting Project

This project demonstrates unit testing in Java using JUnit. 
It contains sample tests and examples to practice and understand testing concepts.

## Technologies
- Java
- JUnit

## Project Structure
- `/src`          -> Source code
- `/tests`        -> Unit tests

## Running Tests
To run the tests, you can use:
```bash
mvn test
```
or your IDE's test runner.

## Objective
- Learn and practice unit testing principles in Java
- Understand how to write testable code
- Apply test-driven development concepts

## Contributing
This repository is primarily for academic purposes.
